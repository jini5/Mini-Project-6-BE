### 금융 상품 쇼핑몰 만들기

- 로그인/회원가입 기능
- 금융상품 정보 (대출 등) 를 검색/리스팅하는 기능 (검색 시 기간 / 내용 등으로 필터링 기능)
- (회원 별) 특정 금융상품을 관심 상품으로 등록/해제하는 기능
- (회원 별) 관심 상품으로 등록해놓은 금융상품만 모아서 보는 기능
- (회원 별) 특정 금융상품을 장바구니에 담는 기능
- 장바구니, 검색, 메뉴 버튼 등을 통해 화면을 전환하는 기능

- Project : Gradle 7.6
- Spring Boot : 2.7.8
- DataBase : MariaDB

- Spring Security
